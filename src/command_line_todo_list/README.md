# Project 1 - Command Line Todo List

## Description

This is Project 1 of my Java foundation projects to 
improve my Java skills and build progressively more 
difficult backend projects

## Features

- Add todo
- Show todo list
- Mark todo as done
- Delete a todo
- Exit program

## Requirements

- Java(JDK)

## How to run

* Project 1 and 2 use the same commands to run

- Navigate to src

- Run
```bash
javac command_line_todo_list/*.java
java command_line_todo_list.Menu
```

## What I learned

- Improved error handling
- Better class creation
- How to approach building a small program
