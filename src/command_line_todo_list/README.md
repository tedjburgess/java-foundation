# Project 1 - Command Line Todo List

## Description

This is project 1 of my java foundation
projects to improve my java skills and build
progressively more difficult backend projects

## Features

- Add todo
- Show todo list
- Mark todo as done
- Delete a todo
- Exit program

## Requirements

- Java(JDK)

## How to run

- Navigate to src

- Run
```bash
javac command_line_todo_list/*.java
java command_line_todo_list.Menu
```
