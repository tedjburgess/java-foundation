Project 0 - Command line user registry

Description: 
This progam was built to start building my java foundation in order to progress to more difficult projects. The program includes simple adding, listing, and searching of users.

Features:
- Add users: includes fields Name, Age, Hometown
- List users: Displays all added users
- Find users: Search for a user by name
- Show menu: Reprints the menu
- Clear terminal: Adds whitespace to improve readablity 
- Exit program: Exit by entering an x

Requirements: 
-Java (JDK)

How to run:
- Navigate to micro/src/user_program
- Compile: javac *.java
- Run: java Menu

What I Learned: 
- Switch statements vs. If else
- What a constructor does and why you need one
- How to think about creating classes
- When to use .equals()
